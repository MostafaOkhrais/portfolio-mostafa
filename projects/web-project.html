About This Project

Manara (منارة) is a full-stack educational platform built specifically for Tawjihi (General Secondary Education) students in Palestine. The project consists of two integrated components: an Android application built with Java and a companion website built with HTML, CSS, and JavaScript — both sharing the same database for seamless content synchronization.

The platform provides students with access to textbooks, past exam papers, summaries, and study materials across all subjects. Content is available for offline access, allowing students to study without an internet connection. The project reflects a deep understanding of the local educational system and the specific needs of Tawjihi students.

Key Features

  • Android application built with Java for mobile access
  • Companion website for desktop and web access
  • Shared database architecture between app and website
  • Full Arabic language support with RTL layout
  • Offline content access — books, exams, and summaries available without internet
  • Clean, modern UI with purple and gold brand identity
  • Responsive design optimized for all devices

Technical Architecture

The project follows a full-stack architecture with a shared database layer powering both the Android app (Java) and the website (HTML/CSS/JS). The website features a modern dark theme with purple and gold accents, Arabic-first RTL design, and a clean user interface optimized for browsing educational content. Version control and project tracking were managed through Git throughout development.

Project Insights

Building Manara required coordinating two separate front-end platforms (Android and web) around a single data layer — a valuable lesson in cross-platform architecture. The project also demanded careful attention to Arabic typography, RTL layout principles, and designing an interface that feels natural and intuitive for the target audience of students. Managing the full stack independently — from database design to UI screens — provided a comprehensive end-to-end development experience.
