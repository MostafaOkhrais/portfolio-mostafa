About This Project

Manara (منارة) is a native Android application built with Java, designed specifically for Tawjihi (General Secondary Education) students in Palestine — currently under active development. The app will provide comprehensive access to textbooks, past exam papers, summaries, and study materials across all subjects, all available for offline use. It will share a unified database with the companion Manara website, ensuring seamless content synchronization across platforms.

The app features a modern Material Design interface with a dark purple and gold brand identity, full Arabic/RTL support, and an intuitive navigation system. Built with a focus on performance and offline reliability, Manara delivers a smooth user experience even in low-connectivity environments.

Key Features

  • Browse and download textbooks, past exams, and summaries by subject
  • Full offline access — all downloaded content available without internet
  • Built-in PDF viewer with smooth navigation and zoom
  • Track and continue where you left off with recent files
  • Bookmark favourite files and save them for quick access
  • Exam countdown timer and study reminders
  • User accounts with cloud sync across devices
  • Full Arabic interface with RTL layout and Arabic typography (Cairo font)
  • Dark theme optimized for reading and study sessions
  • Shared database with the companion Manara website platform

Status

Technical Architecture

The app is built using Java with Android SDK, following a clean architecture with Room database for local storage, Retrofit for network communication, and Data Binding for efficient UI updates. The navigation system uses Android Navigation Component with a bottom navigation bar pattern. The app integrates with a shared backend database that also powers the Manara website, enabling cross-platform data consistency. The project is managed with Git for version control and follows Material Design guidelines for a polished user experience.

Project Insights

Building Manara Android was a comprehensive exercise in native mobile development — covering the full spectrum from database design and offline storage to polished UI/UX with Arabic RTL support. Coordinating the app with a companion website sharing the same data layer provided valuable experience in cross-platform architecture. Implementing features like offline PDF viewing, download management, and progress tracking required careful attention to performance and user experience on mobile devices with varying capabilities.
